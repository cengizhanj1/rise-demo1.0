# Rise

A modern Android discipline and habit tracking application built with Kotlin and Jetpack Compose.

## 📱 Features

- Habit Tracking
- Daily Streak System
- Daily Coach Messages
- Progress Tracking
- Local Data Storage
- Notifications & Reminders
- Home Screen Widget
- Premium Features (Planned)

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- StateFlow
- Material 3
